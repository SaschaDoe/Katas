# Katas

<details>
  <summary>Number Guessing Game</summary>

Time Boxed: 1h
Folowing requirements must be proven! to be fullfilled:
  
1. Message that the game has started when game opened.
2. Input max range of a random number when game opened.
3. Generate a random number between 0 and max range at the beginning of a round.
4. Ask for guess at the beginning of a turn.
5. When guess equals random number, show winning message and start new round.
6. When guess is greater/smaller as random number, show that the rn is greater/smaller and start new turn.
7. The game exits when the player type quit or Quit 
8. The GUI is the standard console but should be interchangeable
